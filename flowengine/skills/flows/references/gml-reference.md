# GML 速查

- 模式：单值或对象构造。单行多项用逗号分隔，>5 项或较长改用多行并对齐 `=`。  
- 赋值/返回：只有一个表达式则为返回值；多语句时由最后一项决定；可用 `return` 显式返回。  
- 字面量：字符串用单引号；模板用反引号与 `${}`；数值/布尔/null 直接写。  
- 逻辑/计算：支持 `+ - * / %`、比较、`&& || !`、三元、`||` 默认值、`CASE ... END`。  
- 作用域：临时变量前缀 `$`，输出不包含；`this` 引用已定义字段/变量避免重复计算。  
- 复合结构：对象 `{ ... }` 支持展开 `...obj`，数组 `[ ... ]`；索引 `arr[0]`、`arr[#]`。  
- 函数：UDF 形式 `FUNC(arg1, arg2)`，可嵌套（如 `MD5(CONCAT(...))`）。
