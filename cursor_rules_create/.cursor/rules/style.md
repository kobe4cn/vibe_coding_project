# MotherDuck 风格规则

- 品牌基调：俏皮但专业，鸭子为核心元素，整体保持温暖亲和与现代科技感。
- 调色板：主色 `#FFD93D`，深蓝 `#1E3A5F`，辅助天蓝 `#D5E8F0`，常用中性色（白、浅灰、文本灰 `#6B7280`、深灰 `#374151`、近黑 `#111827`）；功能色 Success `#10B981` / Warning `#F59E0B` / Error `#EF4444` / Info `#3B82F6`。可用品牌渐变（深蓝系 Hero，黄系 CTA 悬停）。
- 字体：正文用 Inter（sans），代码用 JetBrains Mono（mono）。字号层级：H1 56/36，H2 40/28，H3 28/22，正文 16，正文大 18，正文小 14，Caption 12，行高 1.1–1.7，标题略负字距。
- 间距：8px 网格（4,8,12,16,20,24,32,40,48,64,80,96,128），卡片/输入常用 16–32px，按钮 8–16px（小/中/大）。
- 圆角：优先 8–16px；小元件 4px，大区块/弹窗 16–24px，胶囊/头像用 full。
- 阴影：卡片默认 md，悬停 lg；弹窗/下拉用 xl/2xl；品牌黄/蓝阴影可用于强调。
- 布局：容器最大宽 1280px（可至 1440px），左右内边距 24/48/64；区块 padding 80–120px；Grid 响应式 2–4 列，移动端先单列。
- 响应式断点：sm 640, md 768, lg 1024, xl 1280, 2xl 1536。移动优先，保证触摸友好（最小点击 44px）。
- 组件要点：
  - 按钮：主按钮黄底深色字，悬停提亮并加阴影；次按钮透明+灰边，悬停改黄边；幽灵/链接按钮保持轻量。
  - 卡片：白底、12–16px 圆角、md 阴影，悬停上移 + lg/xl 阴影；定价卡可用 2px 边框与黄强调。
  - 导航：固定顶部，半透明白背景 + 1px 轻边框，链接悬停浅灰；导航 CTA 用品牌黄。
  - 输入：白底、1px 灰边、8px 圆角，聚焦黄边+浅黄光晕；占位符用浅灰。
  - 表格：表头天蓝底，单元格 16–24px 内边距，悬停行浅灰；比较表首列 sticky。
  - 徽章/标签：圆角胶囊，黄/灰/成功绿等轻底色；标签可带小图标间距 6px。
  - FAQ 折叠：大号标题行（18px/600），展开内容 16px 行距 1.7，分隔线浅灰。
- 动效：过渡 150–300ms ease；常用 fadeIn、float、hover-lift 上移与阴影强化，适度使用保持专业。
- 插图：手绘卡通鸭，主体黄、橙嘴脚、柔和阴影与少量渐变；用于 Hero/空状态/装饰，保持温暖友好。
- 代码与文档：代码块深蓝底、圆角 12px、浅灰字，关键字黄、字符串浅绿、注释灰；行内 code 浅灰底、蓝字。

使用时保持留白、清晰层级和技术感，优先移动端体验，突出品牌黄与鸭子元素。***
